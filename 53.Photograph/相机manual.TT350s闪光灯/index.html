📸 TT350s闪光灯使用说明

@tag: #摄影 #闪光灯 #说明书

闪光灯按键：

Mode键：

  • 短按 在Multi(频闪), TTL, M之间切换
  • 长按 进入频闪, set设定 3 - 2Hz, 一共闪3次频率2Hz

Sync键：

  • 短按 进入高速模式, 闪光指数下降
  • 长按 切换无线模式的: 无线主 / 无线从 / 机顶灯

Slave键：

  • 短按 设置无线的分组: A / B / C
  • 长按 设置无线频道

Zoom键

  • 短按 调节焦距（扩散板打开的情况下固定14mm）
  • 长按
    • AF ON（AF辅助闪光）
    • ST ON（自动休眠秒数）
    • BL ON（背光秒数）

X1s 引闪器按键：

CH/OK：

  • 短按 设置频道号0~32
  • 长按 进入Fn设置, Fn00~Fn07, 按GR可以设置：
    • Fn01 设置延迟
    • Fn04 频闪模式
