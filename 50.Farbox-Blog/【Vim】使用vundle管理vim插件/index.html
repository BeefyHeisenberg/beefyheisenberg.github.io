使用 Vundle 管理 Vim 插件（Windowns 7）

分类于 50.FarBox博客备份
字数统计: 277

(1)安装win版本的gvim:

下载地址 http://www.vim.org/download.php
比如安装目录是F:\Program Files (x86)\Vim, 并在此目录下新建”bundle\vundle”目录.

(2)安装git

安装git命令行: http://git-scm.com/ (已被长城)
或者github带GUI的版本: http://windows.github.com/
并且把git所在的目录加进环境变量PATH中.

(3) 安装vundle

打开cmd, 输入以下:

git clone https://github.com/gmarik/vundle “F:\Program Files (x86)\Vim\vimfiles\bundle\vundle”

保证git.exe在你的环境变量中.
